# Axiom – Premium SaaS Landing Page

A production-ready SaaS landing page with sharp minimal aesthetic, editorial typography, and sophisticated scroll-triggered animations. Built with React, Vite, and Framer Motion.

## Design Direction

**Sharp Minimal with Editorial Typography** — a refined, high-contrast aesthetic inspired by modern SaaS leaders.

- **Typography:** Playfair Display (serif headlines) + Inter (body text)
- **Color Palette:** Charcoal (#1a1a1a) + White + Deep Teal (#0d7377) accent
- **Motion:** Scroll-triggered fade-ins, staggered animations, subtle parallax
- **Layout:** Asymmetric feature grid, diagonal dividers, overlapping elements, generous whitespace
- **Atmosphere:** Refined grain texture overlay, minimal custom gradients, premium polish

## Stack

- **Frontend:** React 19 + Vite 6
- **Styling:** Tailwind CSS
- **Motion:** Framer Motion
- **Icons:** Lucide React
- **Components:** Headless UI

## Sections

1. **Sticky Navigation** — Logo, nav links, CTA button with scroll state
2. **Hero** — Large justified headline, subheading, dual buttons, visual preview
3. **Features** — Asymmetric alternating layout with scroll-triggered animations
4. **Pricing** — 3 tiers with featured middle tier, elevated design
5. **Testimonials** — Quote carousel with manual/dot controls
6. **FAQ** — Smooth accordion with icon rotation
7. **Footer** — Multi-column layout, brand info, legal links
